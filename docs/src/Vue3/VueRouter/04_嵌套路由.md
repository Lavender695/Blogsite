# 例子

- `children`属性，是一个数组

> index.js

```javascript
        { 
            path: "/home", 
            component: () => import(/*webpackChunkName: 'home'*/"../Home.vue"),
            children: [
                {
                    path: "",
                    redirect: '/home/recommend'
                },
                {
                    path: "recommend",
                    component: () => import("../HomeRecommend.vue")
                },
                {
                    path: "ranking",
                    component: () => import("../HomeRanking.vue")
                }
            ]
        },
```

> Home.vue

```vue
<template>
    <h2>Home</h2>

    <router-link to="/home/recommend">recommend</router-link>
    <br>
    <router-link to="/home/ranking">ranking</router-link>

    <router-view></router-view>
</template>
```
