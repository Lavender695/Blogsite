# slice和splice

| 特性             | slice                  | splice                                  |
| ---------------- | ---------------------- | --------------------------------------- |
| 作用             | 提取数组的一部分       | 添加/删除数组中的元素                   |
| 是否改变原数组？ | 否（纯函数）           | 是（破坏性方法）                        |
| 返回值           | 新数组，包含提取的元素 | 被删除元素组成的数组                    |
| 参数含义         | [start, end)           | (start, deleteCount, item1, item2, ...) |
